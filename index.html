Introduction

Welcome to StyleFlex, a lightweight CSS framework designed to make building responsive, beautiful websites easy. It combines the power of CSS Grid and Flexbox with pre-built components to jumpstart your development process. Let’s dive into the features that make StyleFlex stand out.

Grid System

StyleFlex includes a 12-column responsive grid system, allowing you to build flexible layouts with ease. Example: 


          <div class="container">
            <div class="row">
              <div class="col-6">50% width</div>
              <div class="col-6">50% width</div>
            </div>
          </div>
  • Classes: col-[size], row, container
  • Breakpoints: xs, sm, md, lg, xl
Flexbox Utilities

Leverage powerful Flexbox utilities to control the alignment and distribution of elements in a flexible way.

Classes:

  • .flex-center: Center content horizontally and vertically.
  • .flex-column: Stack elements vertically.

Example:


          <div class="flex-center">
            <div>Content 1</div>
            <div>Content 2</div>
          </div>
Typography

StyleFlex offers a simple, scalable typography system. Headers:


          <h1>Heading 1</h1>
          <h2>Heading 2</h2>
          <h3>Heading 3</h3>

Body Text:

  • Base font size: 16px
  • Font-family: sans-serif
Colors and Themes

Customize your site’s color scheme using StyleFlex's predefined color palette or set your own with custom variables. Primary Colors:

  • .bg-primary: Primary background color
  • .text-primary: Primary text color

Example:


          <div class="bg-primary text-light">
            Primary Colored Section
          </div>
Buttons and Forms

StyleFlex provides pre-styled buttons and form elements for quick development. Button Styles:

  • .btn-primary, .btn-secondary, .btn-outline

Form Input Example:


          <input type="text" class="form-control" placeholder="Enter text">
          <button class="btn-primary">Submit</button>
Responsive Breakpoints

StyleFlex is mobile-first. Here are the defined breakpoints for responsiveness:

  • xs (up to 576px)
  • sm (576px and up)
  • md (768px and up)
  • lg (992px and up)
  • xl (1200px and up)
Customizing with Variables

StyleFlex allows for easy customization through CSS variables. Change the theme or layout by overriding the default variables. Example: 


          :root {
            --primary-color: #3B7A57;
            --font-family: 'Arial, sans-serif';
          }
Getting Started

To start using StyleFlex, simply link the framework’s CSS file in your HTML file: 


          <link rel="stylesheet" href="styleflex.css">

Or, install via npm:

npm install styleflex
Browser Support

StyleFlex supports the latest versions of all modern browsers, including:

  • Google Chrome
  • Firefox
  • Safari
  • Microsoft Edge
  • Opera
